Color Scheme

input color will be loaded into palette generator once "load palette" button is pressed.
can check by looking at "input_hex" value and "input_hue" value, input_hue gets loaded into the loadcolor() function
Currently you need to slect a color before clicking input, the generator defaults to random monochrome.
you can switch between color schemes after selecting a color, just press load palette each time.
not inputting anything but changing the color scheme will result in random of that type.
